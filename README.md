# clean-bar-win
My desktop setup using Windhawk tweaks and YASB configs.

This repository contains my desktop customization setup using **Windhawk** and **YASB**.  

---

## 🖥️ Tools Used
- [Windhawk](https://github.com/ramensoftware/windhawk) → Windows customization tweaks
- [YASB](https://github.com/amnweb/yasb) → Yet Another Status Bar for Windows

---


## ⚙️ Windhawk Tweaks
I’ve listed the exact Windhawk mods I use along with their JSON configs (from the Advanced tab), so you can copy-paste them directly into your own installation.

### 📌 Mod List
Here are the mods I currently use:
- **Windows 11 Taskbar Styler**
- **Windows Start Menu Styler**
- **Windows 11 Notification Center Styler**
- **Taskbar tray system icon tweaks**
- **Taskbar height and icon size**
- **Taskbar auto-hide when maximized**
- **Customize Windows notifications placement**
- **Taskbar Dock Animation**
- **Taskbar auto-hide speed**

Everything else are optional, you can install and tweak what you like

---

## ⚙️ YASB Installation

   - Install from [YASB GitHub](https://github.com/amnweb/yasb)
   - Copy `yasb/config.yaml` and `yasb/styles.css` into:
     ```
     %userprofile%\.config\yasb
     ```

---

## 🎨 Notes
- Some tweaks may depend on specific Windows builds and resolution i use 2K monitor.  
- Feel free to modify for your own setup!
