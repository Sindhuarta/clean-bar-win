# Clean Desktop Mod
My desktop setup using Windhawk tweaks and YASB configs.

This repository contains my desktop customization setup using **Windhawk** and **YASB**.  
I’m sharing these files so others can replicate or adapt the look and feel of my desktop.


---

## 🖥️ Tools Used
- [Windhawk](https://github.com/ramensoftware/windhawk) → Windows customization tweaks
- [YASB](https://github.com/amnweb/yasb) → Yet Another Status Bar for Windows
- [Coodesker](https://www.coodesker.com/) → Desktop Organizer (optional)

---


## ⚙️ Windhawk Tweaks
I’ve listed the exact Windhawk mods I use along with their JSON configs (from the Advanced tab), so you can copy-paste them directly into your own installation.

### 📌 Mod List
Here are the mods I currently use:
- **Windows 11 Taskbar Styler** - _json included_
- **Windows 11 Start Menu Styler** - _will update, still making better start menu to match the taskbar_
- **Windows 11 Notification Center Styler** - _Translucent Shell for now_
- **Taskbar tray auto-hide (show on hover)**
- **Taskbar height and icon size** - _json included_
- **Taskbar auto-hide when maximized**
- **Customize Windows notifications placement**
- **Taskbar Dock Animation**
- **Taskbar auto-hide speed**

> If a Windhawk mod listed here does not include a JSON configuration, that means I’m using the **default values** provided by the mod.  
> Only the mods with custom settings have their JSON shown in the sections below.

---

## ⚙️ YASB Installation

   - Install from [YASB GitHub](https://github.com/amnweb/yasb)
   - Download `yasb-frosted-glass.zip`
   - extract `yasb/config.yaml` and `yasb/styles.css` into:
     ```
     %userprofile%\.config\yasb
     ```
